# 🌾 KisanMitra – AI-Powered Smart Farming Platform

KisanMitra is an AI-powered smart farming platform designed to help farmers make informed decisions through artificial intelligence and digital technology. The platform connects farmers with agricultural experts while providing essential farming services in one place.

## 🚀 Features

- 🤖 AI Assistant for instant farming support
- 🌱 AI-based Crop Disease Detection using images
- 👨‍🌾 Connect Farmers with Agricultural Experts/Doctors
- 📍 Live Location Sharing for better assistance
- 🌐 Multilingual Support
- 📈 Real-Time Mandi (Market) Rates
- 📞 Easy communication between farmers and experts
- 📱 Simple and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- AI & Machine Learning
- Git & GitHub

## 🎯 Project Objective

The goal of KisanMitra is to bridge the gap between farmers and modern technology by providing AI-driven solutions that improve crop health, increase productivity, and make expert guidance easily accessible.
